Artificial Neural
Networks

The building block of all deep learning. Understand how a network of simple mathematical units can learn any function — and why that's remarkable.

What is an Artificial Neural Network?

An ANN is a computational model inspired by the biological brain. It consists of layers of interconnected nodes (neurons) that transform input data into output predictions by learning patterns from examples.

The key insight is this: any function can be approximated by a sufficiently large neural network. This is called the Universal Approximation Theorem.

Core idea A neural network is just a chain of matrix multiplications followed by non-linear functions. Training it means adjusting the numbers in those matrices until the output matches what we want.

Architecture Overview

Every ANN has at least three kinds of layers:

Input Layer

Receives raw data — pixels, numbers, embeddings. No computation, just representation.

Hidden Layers

Where the learning happens. Each layer extracts increasingly abstract features from the previous layer.

Output Layer

Produces the final prediction — a class probability, a number, a label.

A fully-connected ANN with 3 inputs, 2 hidden layers, and 2 outputs

The Neuron — One Unit

Each neuron does three things: receives weighted inputs, sums them up (with a bias), then applies an activation function to produce an output.

output = activation( Σ(wᵢ · xᵢ) + b )
xᵢ

Inputs — numbers coming from the previous layer or raw data.

wᵢ

Weights — learned parameters that scale each input's contribution.

b

Bias — a learnable offset, shifts the activation threshold.

activation

Non-linear function (ReLU, sigmoid…) — without this, the whole network collapses to a single linear function.

Analogy Think of a neuron as a voting booth. Each input votes with a certain weight. The bias is the minimum score needed to fire. The activation decides if the result crosses the threshold.

Forward Pass

The forward pass is simply running data through the network from left to right — input layer → hidden layers → output. Each layer transforms the data.

Layer 1: h¹ = activation(W¹ · x + b¹) Layer 2: h² = activation(W² · h¹ + b²) Output: ŷ = softmax(W³ · h² + b³)

In matrix form, each layer is just: h = σ(Wx + b). Training means finding the right values of W and b.

Why layers? Each layer learns a different level of abstraction. Early layers detect simple patterns (edges, numbers), later layers combine them into complex concepts (faces, words).

Loss Function

Once we have an output ŷ, we compare it to the true label y using a loss function. The loss quantifies how wrong the network is.

MSE

Mean Squared Error — for regression tasks: L = (1/n) Σ(y − ŷ)²

Cross-Entropy

For classification: L = −Σ y·log(ŷ). Penalises confident wrong predictions heavily.

Backpropagation

Backprop is how a network learns. It propagates the error signal backwards through the network, computing how much each weight contributed to the mistake — then adjusts them.

The chain rule Backprop is just the chain rule of calculus applied repeatedly. The gradient of the loss w.r.t. any weight is the product of gradients along the path from that weight to the output.
∂L/∂W = ∂L/∂ŷ · ∂ŷ/∂h · ∂h/∂W Weight update: W ← W − η · ∂L/∂W
η (eta)

Learning rate — how large a step to take each update. Too large → overshoot. Too small → slow.

Gradient

Direction of steepest ascent in loss space. We step in the negative direction to descend.

Epoch

One full pass of the entire training dataset through forward + backward pass.

Mini-batch

Training on small subsets (e.g. 32 samples) rather than all data at once — more efficient.

Activation Functions

Without activation functions, stacking layers adds no expressive power — they'd all collapse into a single linear transformation. Non-linearities are what let networks learn complex patterns.

Sigmoid

σ(x) = 1/(1+e⁻ˣ) → output in (0,1). Used in output layers for binary classification. Suffers from vanishing gradients.

Tanh

tanh(x) → output in (−1,1). Zero-centered — often better than sigmoid for hidden layers.

ReLU

f(x) = max(0, x). Default choice for hidden layers. Fast, no vanishing gradient for positive inputs.

Softmax

Converts a vector of scores into a probability distribution summing to 1. Standard output for multi-class classification.

Rule of thumb Use ReLU (or LeakyReLU) for hidden layers. Use Sigmoid for binary output, Softmax for multi-class output. Avoid Sigmoid/Tanh in deep hidden layers — they saturate and kill gradients.
