Neuron Network Basics

A Neural Network is a computing system inspired by how the human brain works. It is made up of connected units called neurons that process information.

What is a Neuron?

A neuron is the smallest unit of a neural network. It receives inputs, processes them, and produces an output.

Parts of a Neuron

  • Input – Data given to the neuron
  • Weight – Importance of each input
  • Bias – Helps adjust the output
  • Activation Function – Decides the final output

How a Neural Network Works

  1. Inputs are given to neurons
  2. Each input is multiplied by a weight
  3. All values are added together with bias
  4. Activation function produces output
  5. Output is passed to next layer

Layers in a Neural Network

  • Input Layer – Receives data
  • Hidden Layer – Processes data
  • Output Layer – Gives final result

Simple Example

Neural networks are used in:

  • Image recognition
  • Speech recognition
  • Spam detection
  • Recommendation systems

Conclusion: Neural networks learn from data and improve over time, just like the human brain learns from experience.
